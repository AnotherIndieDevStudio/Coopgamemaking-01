What is this?

If you have stumbled here by accident, then welcome. This subreddit is dedicated to making a game! How cool is that? This is a social experiment to see where the community takes game creation. Players can make it an extremely fun game or make it completely rubbish.

It's up to you guys to come up with the ideas and the code for the game. No we're not lazy. We just want to see where this is going.

The current game we are working on is simply a prototype and is just a testing grounds to see whether the community is will to participate. Whilst written in Javascript, we feel that allot of content can be added.

Later, if this proves to be successful, we can let the real fun begin and start working with proper game engines where the community can attempt to make a massive hit game.

What do I need to participate?

Nothing really. The current game is using Javascript with Jquery. Javascript is really easy to pickup on and you can create some cool content with minimal work. The game could also use some graphics, if that is what the community feels it needs. You could work with the HTML and CSS to make it a bit more stylish if you feel the need to as well.

Making The Game (How To):

Submission

Submit your idea with a screenshot (if relevant), code uploaded to Pastebin or Github (fully commented and readable), resources (images, sounds) uploaded to a respectful file sharing service. Then add a description to the idea as well as a TLDR at the end. Use the flair "Submission".

Removal Request

If you want something removed, link to the thread in which the content is. Leave a valid reason as to why it should be removed from the game. Use the flair "Removal Request".

Idea

If you have an idea but aren't familiar with programming, use the flair "Idea" with a good description of what you want.

Working On

If you are working on something don't forget to let people know using the "Working On This" flair. This way, it prevents any conflicts.

How does my code get included

Any ideas that are highly upvoted will be added. Also anything we personally feel needs to be added will also be added assuming that they aren't drastically downvoted.
